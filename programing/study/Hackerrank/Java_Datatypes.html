Java

Javaには主なデータタイプとして,char,boolean,bute,short,int,long,float,doubleがあります. 主に使うデータタイプとして,byte,short,int,longがあります.
byte 8bit
short 16bit
int 32bit
long 64bit

整数で入力があたえられたとき,どのデータタイプが使えるかきめなくてはいけません.
